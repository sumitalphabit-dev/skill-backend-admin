=========================================
      ALPHABIT SKILL BACKEND APIs
=========================================
Base URL: http://localhost:5000/api

--- 1. AUTHENTICATION & USERS ---

Register a new user
- Method: POST
- URL: /users/register
- Headers: Content-Type: application/json
- Body (JSON):
{
  "name": "Admin User",
  "email": "admin@example.com",
  "password": "password123"
}

Login User
- Method: POST
- URL: /users/login
- Headers: Content-Type: application/json
- Body (JSON):
{
  "email": "admin@example.com",
  "password": "password123"
}
*Note: Returns a "token". Use this token in the "Authorization" header for Admin routes.*


--- 2. PUBLIC BLOG ROUTES ---

Get All Blogs (Public)
- Method: GET
- URL: /blogs
- Headers: None

Get Single Blog By Slug (Public)
- Method: GET
- URL: /blogs/:slug
- Headers: None
- Example: /blogs/my-first-blog


--- 3. PROTECTED ADMIN BLOG ROUTES ---
*Note: All routes below require the Authorization header.*
*Header -> Key: Authorization, Value: Bearer <your_jwt_token>*

Get All Blogs (Admin)
- Method: GET
- URL: /admin/blogs

Get Single Blog By ID (Admin)
- Method: GET
- URL: /admin/blogs/:id
- Example: /admin/blogs/64a2b3c4d5e6f7g8h9i0j1k2

Create New Blog (Admin)
- Method: POST
- URL: /admin/blogs
- Body (form-data):
  - title (text): "My New Blog"
  - content (text): "This is the content."
  - slug (text): "my-custom-slug" [Optional]
  - image (file): Select an image file

Update Blog By ID (Admin)
- Method: PUT
- URL: /admin/blogs/:id
- Body (form-data):
  - title (text): "Updated Title"
  - content (text): "Updated content."
  - slug (text): "updated-slug" [Optional]
  - image (file): Select an image file [Optional]

Delete Blog By ID (Admin)
- Method: DELETE
- URL: /admin/blogs/:id

--- 6. SEMINAR REGISTRATION ROUTES ---

Submit New Seminar Registration (Public)
- Method: POST
- URL: /seminars/register
- Headers: Content-Type: application/json
- Note: Rate limit max 5 requests per 10 minutes per IP. Same-phone signup prevented for duplicate batch.
- Body (JSON):
{
  "name": "Jane Doe",
  "phone": "9876543210",
  "email": "janedoe@example.com",
  "college": "Test University",
  "course": "Computer Science",
  "city": "Mumbai",
  "source": "Instagram",
  "seminarBatch": "July 2026 Batch"
}

Get All Seminar Registrations (Admin)
- Method: GET
- URL: /admin/seminars?batch=July 2026 Batch&status=registered&search=Jane&page=1&limit=10
- Headers: Key: Authorization, Value: Bearer <your_jwt_token>

Get Single Seminar Registration By ID (Admin)
- Method: GET
- URL: /admin/seminars/:id
- Headers: Key: Authorization, Value: Bearer <your_jwt_token>

Update Seminar Registration Status By ID (Admin)
- Method: PUT
- URL: /admin/seminars/:id
- Headers: Key: Authorization, Value: Bearer <your_jwt_token>, Content-Type: application/json
- Body (JSON):
{
  "status": "attended"
}

Delete Seminar Registration By ID (Admin)
- Method: DELETE
- URL: /admin/seminars/:id
- Headers: Key: Authorization, Value: Bearer <your_jwt_token>

Export Seminar Registrations to CSV (Admin)
- Method: GET
- URL: /admin/seminars/export?batch=July 2026 Batch&status=registered
- Headers: Key: Authorization, Value: Bearer <your_jwt_token>

Get Seminar Registration Stats (Admin)
- Method: GET
- URL: /admin/seminars/stats
- Headers: Key: Authorization, Value: Bearer <your_jwt_token>


--- 7. SEMINAR EVENT ROUTES ---

Get Currently Active Seminar Event (Public)
- Method: GET
- URL: /seminars/current
- Headers: None
- Note: Returns currently active event (isActive: true) plus calculated `seatsLeft` and `registeredCount`.

Get All Seminar Events (Admin)
- Method: GET
- URL: /admin/seminar-events
- Headers: Key: Authorization, Value: Bearer <your_jwt_token>

Get Single Seminar Event By ID (Admin)
- Method: GET
- URL: /admin/seminar-events/:id
- Headers: Key: Authorization, Value: Bearer <your_jwt_token>

Create New Seminar Event (Admin)
- Method: POST
- URL: /admin/seminar-events
- Headers: Key: Authorization, Value: Bearer <your_jwt_token>
- Body (form-data):
  - batchId (text): "Aug-2026"
  - topic (text): "Cracking Your First Tech Career"
  - date (text): "2026-08-02"
  - startTime (text): "4:00 PM"
  - endTime (text): "6:00 PM"
  - mode (text): "Offline"
  - venue (text): "Alphabit Skill Studio, Rajkot"
  - venueMapLink (text): "https://maps.google.com/?q=..." [Optional]
  - language (text): "English & Gujarati/Hindi mix" [Optional]
  - totalSeats (text/number): "100" [Optional, Default: 100]
  - isActive (text/boolean): "true" [Optional, Default: true]
  - venueImage (file): Select venue image file [Optional]

Update Seminar Event By ID (Admin)
- Method: PUT
- URL: /admin/seminar-events/:id
- Headers: Key: Authorization, Value: Bearer <your_jwt_token>
- Body (form-data):
  - topic (text): "Updated Seminar Topic" [Optional]
  - date (text): "2026-08-10" [Optional]
  - startTime (text): "5:00 PM" [Optional]
  - endTime (text): "7:00 PM" [Optional]
  - totalSeats (text/number): "120" [Optional]
  - isActive (text/boolean): "true" [Optional]
  - venueImage (file): Select new venue image file [Optional]

Delete Seminar Event By ID (Admin)
- Method: DELETE
- URL: /admin/seminar-events/:id
- Headers: Key: Authorization, Value: Bearer <your_jwt_token>

Activate Seminar Event (Admin)
- Method: PATCH
- URL: /admin/seminar-events/:id/activate
- Headers: Key: Authorization, Value: Bearer <your_jwt_token>
- Note: Sets target seminar event isActive: true and deactivates all others.
